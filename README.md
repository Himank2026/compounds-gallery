Compound Gallery is a full-stack web application for managing and displaying chemical compound data.

What I Built

Developed a RESTful backend using Node.js, Express, MySQL, and Sequelize

Designed a relational database schema to store compound details such as name, image URL, description, and metadata

Implemented CRUD APIs with pagination, validation, and proper error handling

Built a frontend using Angular (TypeScript) with a component-based architecture

Implemented a paginated gallery view to display compounds as interactive cards

Created detail, create, edit, and delete views for compounds

Used Angular routing for navigation and services for API communication

Added form validation, loading states, and image previews

Tech Stack

Backend: Node.js, Express, MySQL, Sequelize

Frontend: Angular, TypeScript

Tools: Git, MySQL Workbench, VS Code

Outcome

The application allows users to browse, view, add, update, and delete chemical compounds through a clean UI backed by a structured relational database and REST API.
